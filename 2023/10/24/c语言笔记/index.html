声明和定义


ertern int i和int i的区别
在C语言中extern int i; 是一个声明，它告诉编译器i是一个整数类型的变量，并且其定义在其他地方。这意味着i的内存分配并不发生在此声明处，而是在程序的其他地方。这通常用于当多个文件需要访问同一个全局变量时。

例如，假设你有两个文件file1.c和 file2.c。在 file1.c 中，你定义了一个全局变量int i = 0。然后在file2.c中，你想要访问这个在 file1.c 中定义的变量i，你就可以在 file2.c 中使用 `extern int i;来声明这个变量，这样，file2.c就可以访问和修改file1.c中定义的i了。

总的来说，extern关键字允许在一个文件中访问在另一个文件中定义的全局变量或函数。

注意，声明仅仅是告诉编译器这里有个 变量，但没有给变量分配内存空间。
